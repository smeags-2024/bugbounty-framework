# Installation Notes:
# 1. Open Burp Suite
# 2. Go to Extender > BApp Store
# 3. Search for each extension by name
# 4. Click "Install"
# 5. Restart Burp Suite if prompted

# Manual Installation (for extensions not in BApp Store):
# 1. Download .jar file from GitHub
# 2. Extender > Extensions > Add
# 3. Select extension type (Java)
# 4. Load the .jar file
